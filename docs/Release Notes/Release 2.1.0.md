# Note 2.1 - May 2026

- **Create Global Condition Changes** - Apply a [condition change line](../CarePlan/Graphs.md/#condition-changes) to all target and behavior graphs by toggling on “Apply to all graphs” on a condition line. Delete a global condition line from one or all graphs.

- **Collect Zero on a Skill Acquisition Count Target within a session** - If a targeted skill was not demonstrated, you now have the option to Confirm Zero as the score so a zero is graphed for that target that day. Access this option by opening the [count](../CarePlan/FormsOfMeasure.md/#count) target on the observations tab of a direct care session, during or after the session.

- **New Phase: Generalization** - Set targets to the Generalization phase for clear tracking of when the target is being generalized across settings, timings, people, etc.
    - *Note: Behavior technicians will not be able to see targets at the Generalization phase until updating to Note 2.1.*

- [Prompt levels auto-fade](../CarePlan/ConfigureTargets.md/#configure-a-prompting-workflow-on-a-target) properly based on mastery over consecutive sessions.

- Session location visibility enhancements.
